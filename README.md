# Perancangan web untuk kegiatan Open House Program Pemasaran dan Admisi Telkom University

Open House merupakan strategi pemasaran dengan membuka akses kepada masyarakat umum untuk melihat atau mengikuti kegiatan didalam lingkungan penyelenggara oleh suatu institusi. Dalam hal ini PADMI (Program Pemasaran dan Admisi) Telkom University menerapkan strategi ini dalam upaya untuk mempromosikan dan mengenalkan Telkom University kepada masyarakat luas. Dari hal tersebut PADMI memerlukan sistem website untuk mendukung program ini. pembuatan modul backend–frontend, hingga integrasi database dan fitur keamanan. Aplikasi dikembangkan secara iteratif berdasarkan masukan tim riset agar memenuhi kebutuhan operasional Open House dan dapat digunakan sebagai solusi yang stabil, efisien, dan mudah dioperasikan. 

Fitur-Fitur : 
1.) User : Login dan Register, Dashboard User, Scan booth, point dan klaim hadiah
2.) Admin super : Dasboard admin, tambah staff dan booth, pengaturan syarat klaim, export data ke format excel
3.) Admin Staff : Dashboard staff, scan presensi dan scan qr hadiah, konfirmasi presensi dan klaim hadiah, dan mendaftarkan peserta ke campus tour OTS.

Aplikasi ini nanti nya akan benar benar di gunakan oleh PADMI guna mendukung program Open House.
